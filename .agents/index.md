## Quick Summary

**DataLinq.NET** is a .NET data processing framework built on the philosophy: *"LINQ all the way down."* It provides a unified API for processing data from files (CSV/JSON/YAML), databases (EF Core), message queues (Kafka), and cloud platforms (Spark, Snowflake) — all using the same LINQ operators with constant memory consumption.

The framework is organized around the **SUPRA pattern** (Sink → Unify → Process → Route → Apply) and features a declarative **Cases pattern** for routing without nested if/else. Premium features include LINQ-to-Spark and LINQ-to-Snowflake translators.
